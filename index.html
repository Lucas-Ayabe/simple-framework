Simple Framework

Componentes simples para compor o seu projeto complexo

Introdução

Este framework tem como propósito ser uma coleção de componentes simples, variados e adaptáveis para vários tipos de projetos.

Características

Escrito em Sass

Este framework foi escrito com o poderoso pré-processador de CSS, usando sua sintaxe de SCSS, desta maneira, seus componentes são fácilmente editaveis atráves de variáveis.

Aplicado a metodologia BEM

Para que os componentes fossem escaláveis e o framework pudesse oferecer uma API consistente, todos os componentes foram escritos na metodologia BEM.

Código bem documentado

Todos os estilos do framework possuem uma documentação própria dentro de cada arquivo, para facilitar um primeiro contato com o framework.

Componentes

Nossos componentes são divididos em duas categorias:

  • Objetos
  • Componentes

Objetos são todos os estilos relacionados a layout e possuem o prefixo o- em suas classes.

E temos os componentes visuais em si, como cards, botões e etc. Estes últimos possuem o prefixo c- em suas classes.
